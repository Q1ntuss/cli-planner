# Консольна утиліта «Трекер проєктів» (Варіант №22)

Проєкт виконано в межах навчальної практики з розробки програмного забезпечення.  
Виконавець: Сасін Павло, студент групи ІПЗ-21.

---

## 🎯 Функціонал програми

- Додавання проєктів із унікальним ID та дедлайном.
- Додавання задач до проєктів із назвою та описом.
- Перегляд усього каталогу проєктів у вигляді консольної таблиці зі статусом та списком задач.
- Перегляд задач конкретного проєкту.
- Визначення прострочених проєктів (порівняння з поточною датою).
- Видалення проєктів (разом із задачами) з підтвердженням.
- Видалення окремих задач з підтвердженням.
- Збереження даних у текстовий файл.
- Завантаження даних з текстового файлу.
- Автоматичне завантаження даних при запуску (з `default.txt`).
- Автоматичне збереження даних при виході (у `default.txt`).

---

## 🛠 Інструкція із запуску

Для роботи програми потрібен компілятор C++17 (g++, clang, або MSVC).

### Компіляція та запуск:

1. Завантажте файл `main.cpp` у робочу директорію.
2. Відкрийте термінал або командний рядок у цій папці.
3. Скомпілюйте програму командою:

   ```bash
   g++ -std=c++17 main.cpp -o project_tracker
